Detailed Description

Server matchmaking, sessions, and dedicated server management.

This module provides server multiplayer services:

  • Server session management
  • Server matchmaking operations
  • Matchmaking V2 for dedicated servers
  • Session browser operations
  • Dedicated Server Manager (DSM) integration
  • Dedicated Server Hub communication
  • Server QoS management
  • Armada Management Service (AMS) for server fleets
