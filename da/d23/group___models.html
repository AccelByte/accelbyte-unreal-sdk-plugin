Detailed Description

Data structures, request/response models, and DTOs.

This module contains all data structures used throughout the SDK:

  • Request models (API call parameters)
  • Response models (API responses)
  • Event models (WebSocket events)
  • Enumerations and constants
  • Data transfer objects (DTOs)

Models are organized by feature area and correspond to the API modules.
