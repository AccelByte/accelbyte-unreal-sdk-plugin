Overview

The AccelByte Unreal SDK is a plugin for Unreal Engine that acts as a bridge between your game and AccelByte Gaming Services (AGS), providing easy access to backend services for your Unreal Engine game.

Getting Started

For setup and implementation guidelines, visit our Developer Portal.

Key Features

  • Authentication & User Management - User login, registration, and profile management
  • Social Features - Friends, parties, chat, and social interactions
  • Matchmaking - Session management, matchmaking, and server browser
  • Ecommerce - In-game store, inventory, wallet, and entitlements
  • Cloud Save - Save player data to the cloud
  • Leaderboards & Achievements - Track player progress and rankings
  • Groups & Clans - Player communities and guild management
  • User Generated Content - UGC creation and management
  • Analytics & Telemetry - Game analytics and player behavior tracking
  • Multiplayer Services - Session management, P2P networking, and dedicated servers
