Detailed Description

Session management, matchmaking, dedicated servers, and multiplayer infrastructure.

This module provides multiplayer game services:

  • Game session creation and management
  • Matchmaking (skill-based, rule-based)
  • Session browsing and discovery
  • Dedicated server integration
  • Server registration and heartbeat
  • Quality of Service (QoS) for server selection
  • TURN server for NAT traversal
  • Armada Management Service (AMS) for fleet management
